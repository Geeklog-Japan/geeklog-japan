CustomMenu Editor プラグイン

CustomMenu Editor プラグインは，メインメニュー（ヘッダーメニュー）の設定を管理画面で簡単に行えるようにするものです。

設定

CustomMenu Editor プラグインの設定は、「管理者用メニュー - コンフィギュレーション - カスタムメニュー」から変更できます。

カスタムメニューのメイン設定

項目(変数) デフォルト 説明
メニューアイテム保存後の画面遷移(aftersave) カスタムメニュー管理を表示する(list) メニューアイテムを保存した後にどのページへ移動するかを設定します。
  • カスタムメニュー管理を表示する(list): 管理者用のカスタムメニューのリストを表示します（デフォルト）。
  • ホームを表示する(home): サイトのホームページ（トップページ）を表示します。
  • 管理画面トップを表示する(admin): 管理画面を表示します。
メニューレンダラ(menu_render) Geeklogシステム標準(pulldown) メニューをどの方法でレンダリングするかを設定します。
  • Geeklogシステム標準(standard): Geeklogシステム標準の方法でレンダリングします。
  • 階層メニュー対応(pulldown): カスタムメニュー独自の方法でレンダリングします。多階層メニュー（プルダウンメニュー）の表示が可能です（デフォルト）。
IDに付加するプレフィックス(prefix_id) 'cmitem-' ページのソース(HTML)における、メニューアイテムの<li>タグのID属性に付加する文字列を設定します。メニューレンダラが「階層メニュー」であるときにのみ有効です。
