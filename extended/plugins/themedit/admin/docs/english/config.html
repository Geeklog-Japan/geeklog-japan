Resync method Resync manually When you add/remove a theme to/from "Themes allowed to be edited", or a template file to/from "Files allowed to be edited", the Theme Editor plugin will detect it automatically.
  • 'Resync Automatically' -- the plugin will update the data stored in databse automatically.
  • 'Resync manually' -- the plugin will display the information and 'UPDATE database' button.
  • 'Do not resync' -- the plugin will do nothing about the change.
