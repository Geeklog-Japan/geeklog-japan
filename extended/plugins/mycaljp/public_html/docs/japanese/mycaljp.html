Geeklog Mycaljp プラグイン ドキュメント

Geeklog Mycaljpプラグインの設定オプションについて以下に説明します。 インストール手順については、/path/to/geeklog/public_html/admin/plugins/mycaljp/ディレクトリ内のreadme_jp.htmlファイルをお読みください。

設定

Mycaljpプラグインの設定は、「管理者専用メニュー - コンフィギュレーション - Mycaljp」から変更できます。

Mycaljpのメイン設定

項目(変数) デフォルト 説明
headertitleyear 'Y年' ヘッダータイトルのうち、年を示す部分のフォーマットを設定します。
'Y' 年．4 桁の数字．(例: 1999または2003)
'y' 年．2 桁の数字．(例: 99 または 03)
headertitlemonth 'm月' ヘッダータイトルのうち、月を示す部分のフォーマットを設定します。
'm' 月．数字。先頭にゼロをつける．(01 から 12)
'n' 月．数字。先頭にゼロをつけない．(1 から 12)
'F' 月．フルスペルの文字．(January から December)
'M' 月．3 文字形式．(Jan から Dec)
titleorder 年 月(true) ヘッダータイトルの年と月の表示順序を設定します。
一般的に、和風にしたいときは「年 月」(true)、洋風にしたいときは「月 年」(false)を選択します。
sunday '日' 日曜日の文字列を設定します。
monday '月' 月曜日の文字列を設定します。
tuesday '火' 火曜日の文字列を設定します。
wednesday '水' 水曜日の文字列を設定します。
thursday '木' 木曜日の文字列を設定します。
friday '金' 金曜日の文字列を設定します。
saturday '土' 土曜日の文字列を設定します。
showholiday はい(true) 土・日・休日を色分け表示の有無を設定します。
「はい」(true)を選択すると色分け表示します。
「いいえ」(false)を選択すると色分け表示しません。
checkjpholiday はい(true) 日本の休日を調べるかどうかを設定します。
「はい」(true)を選択すると日本の休日を色分け表示の対象とします。
「いいえ」(false)を選択すると日本の休日を色分け表示の対象としません。
enablesrblocks はい(true) 日付のリンクをクリックしたあとの、コンテンツの一覧表示画面において、右サイドバーを表示するかどうかを設定します。
「はい」(true)を選択すると右サイドバーを表示します。
「いいえ」(false)を選択すると右サイドバーを表示しません。
showstoriesintro はい(true) 日付のリンクをクリックしたあとの、コンテンツの一覧表示画面において、記事のイントロ部を表示するかどうかを設定します。
「はい」(true)を選択すると記事のイントロ部を一覧表示します。
「いいえ」(false)を選択すると記事のタイトルを一覧表示します。
use_theme いいえ(false) テーマで提供されているテンプレートを適用するかどうかを設定します。
<public_html>/layout/テーマ名/mycaljp ディレクトリ内のテンプレートを使用するかどうか設定します。
template Default サイトカレンダーの表示に用いられるテンプレートを選択します。
本プラグインには最初から数種類のテンプレートが用意されています。
なお、名前に「_multi_language」を含むテンプレートはマルチランゲージ環境用に設定されたものです。
date_format '[Y-m-d]' サイトカレンダーの日付などをクリックした際の検索結果で用いられる日付の形式を設定します。
「日付の形式」で使用するパラメーターは、PHPのdate()関数のformatパラメーターと同じです。
supported_contents - サイトカレンダーが検索対象としてサポートするコンテンツを設定します。
enabled_contents - サイトカレンダーが検索するコンテンツを設定します。
1 : 有効
0 : 無効

静的ページの設定

項目(変数) デフォルト 説明
sp_type センターブロックに表示されるページのみ(1) サイトカレンダーが検索する静的ページの種類を設定します。
0: すべて
1: センターブロックに表示されるページのみ
2: センターブロックに表示されないページのみ
sp_except 'formmail' サイトカレンダーの検索対象から除外する静的ページのIDを設定します。

ブロックの設定

項目(変数) デフォルト 説明
有効(block_enable) はい(true) ブロックを有効化します。
左ブロックで表示する(block_isleft) はい(1) 左ブロックで表示(1) あるいは右 (0)に表示します。
ブロックの順番(block_order) 10 ブロックカラムのソート番号を指定します。
話題オプション(block_topic_option) すべて(all) 'すべて'なら、すべてのページで表示。'ホームページのみ'ならホームページのみで表示。'話題を選択する'なら選択された話題でのみ表示されます。
話題(block_topic) - 話題オプションが '話題を選択する'のときだけ表示されます。
グループ(block_group_id) Mycaljp Admin パーミッションで使われるグループ
