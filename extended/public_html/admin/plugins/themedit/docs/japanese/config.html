Geeklogテーマエディタプラグイン

設定

主要設定

項目 初期値 説明
すべてのテーマの編集を許可する いいえ 「はい」にすると、「編集を許可するテーマ」の値とは無関係に、すべてのテーマを自動的に利用可能にします。
すべてのファイルの編集を許可する いいえ 「はい」にすると、「編集を許可するファイル」の値とは無関係に、すべてのテーマに関係するファイル(*.thtml, *.css)を自動的に利用可能にします。
編集を許可するテーマ professional このプラグインを使用して編集するテーマ名（大文字小文字を区別します）を設定します。
編集を許可するファイル 'style.css', 'custom.css', 'custom.sample.css', 'style_forum.css', 'header.thtml', 'footer.thtml', 'leftblocks.thtml', 'blockheader-left.thtml', 'blockfooter-left.thtml', 'rightblocks.thtml', 'blockheader-right.thtml', 'blockfooter-right.thtml', 'storytext.thtml', 'featuredstorytext.thtml', 'archivestorytext.thtml', 'article/article.thtml', 'article/printable.thtml', 'menuitem.thtml', 'menuitem_last.thtml', 'menuitem_none.thtml', 'list.thtml', 'listitem.thtml', 'loginform.thtml', 'profiles/contactuserform.thtml', 'profiles/contactauthorform.thtml', 'preferences/profile.thtml', 'users/profile.thtml', 'search/searchform.thtml', 'search/searchresults.thtml', 'submit/submitevent.thtml', 'submit/submitloginrequired.thtml' このプラグインを使用して編集するファイル名（大文字小文字を区別します）を設定します。テンプレートファイル(*.thtml)とカスケィディングスタイルシートファイル(*.css)を指定できます。
データベースの同期方法 手動で同期 「編集を許可するテーマ」にテーマを、「編集を許可するファイル」にファイルを追加・削除する度に、テーマエディタプラグインは自動的に検出します。
  • '自動で同期' -- プラグインはデータベースに保存されているデータを更新します。
  • '手動で同期' -- プラグインは更新情報と、「データベースを更新」ボタンを表示します。
  • '同期しない' -- プラグインは変更を無視します。
ファイルのアップロードを許可する はい 画像ファイルのアップロードを許可するかどうかを決めます。
サムネール画像の幅（単位：ピクセル） 120 サムネール画像の幅（ピクセル）を指定します。
サムネール画像の高さ（単位：ピクセル） 100 サムネール画像の高さ（ピクセル）を指定します。
サムネール画像の列数 6 1行中に表示するサムネール画像の数を指定します。
アップローできるファイルサイズの上限（単位：バイト） 1048576 (= 1MB) アップロードできる画像ファイルの最大サイズ（単位：バイト）を指定します。
